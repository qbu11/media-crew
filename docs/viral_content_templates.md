# 垂类爆款内容模板库

> 本文档汇总了 6 大中文平台的垂类爆款内容模板，供选题研究员和内容创作者 Agent 使用。
> 核心逻辑：**什么火就发什么**

## 一、小红书垂类爆款模板

### 1.1 爆款核心公式

**TDC模型**：目标人群 + 可信解决方案 + 数字化结果
- 示例："混油敏皮 + FDA认证成分，30天逆袭"

**金字塔架构**：权威认证 + 场景拆解 + 效果展示

### 1.3 正文结构模板（5种）

#### 干货攻略类结构
```
【开头】痛点共鸣 + 承诺价值
【正文】
  1. 方法一（步骤+原理）
  2. 方法二（步骤+原理）
  3. 方法三（步骤+原理）
【结尾】总结 + 互动引导
```

#### 种草测评类结构
```
【开头】使用场景/痛点 + 产品引入
【正文】
  - 产品外观/质地
  - 使用感受（分点）
  - 效果对比（图文）
【结尾】适合人群 + 购买建议
```

#### 避坑合集类结构
```
【开头】踩坑经历 + 引发共鸣
【正文】
  ❌ 误区一 + ✅ 正确做法
  ❌ 误区二 + ✅ 正确做法
  ❌ 误区三 + ✅ 正确做法
【结尾】总结 + 评论区互动
```

#### 反差反转类结构
```
【开头】"别再X了！" 制造悬念
【正文】
  - 之前错误做法
  - 发现问题
  - 转变/新方法
  - 效果对比
【结尾】核心观点 + 呼吁行动
```

#### 情感类结构
```
【开头】场景描写 + 情绪共鸣
【正文】
  - 故事展开
  - 心理描写
  - 觉醒/转变
【结尾】金句升华 + 互动
```

### 2.2 完整脚本结构

#### 知识口播类（五段式，60秒）
```
【0-5秒】钩子开场（制造认知缺口）
【5-15秒】问题引入（为什么重要）
【15-50秒】核心内容（3-5个要点）
【50-55秒】总结升华
【55-60秒】互动引导（点赞/关注）
```

#### 剧情类（60秒）
```
【0-3秒】制造悬念/冲突
【3-15秒】铺垫背景
【15-30秒】冲突升级
【30-45秒】高潮反转
【45-60秒】结局升华
```

#### 种草类（30秒）
```
【0-3秒】抛出痛点
【3-10秒】引发共鸣
【10-20秒】产品展示+效果
【20-30秒】促单行动
```

### 2.3 高转化话术公式

**SCQA模型**（麦肯锡）：
- S(Situation) - 交代背景："作为一个{身份}..."
- C(Complication) - 揭示问题："我发现{痛点}"
- Q(Question) - 抛出疑问："为什么{问题}？"
- A(Answer) - 给出方案："今天教你{解决方案}"

**HVA公式**：
- Hook(钩子)："别划走！"
- Value(价值)："这个方法让你{效果}"
- Action(行动)："赶紧试试！"

## 三、知乎垂类爆款模板

### 3.1 爆款公式

```
爆款 = 共鸣 + 干货 + 颠覆认知 + 争议 + 悬念
```

### 3.2 开头模板

#### 盐选专用黄金三步法
```
第一步：美好表象（100字内）- 建立代入感
第二步：致命异常（100字内）- 埋下冲突种子
第三步：行动悬念（100字内）- 迫使读者翻页
```

#### 四种通用开头模板

| 类型 | 结构 | 适用场景 |
|------|------|----------|
| 甜蜜关系+隐藏杀机 | 表面美好 → 暗藏危险 | 病娇、悬疑、复仇 |
| 日常场景+灵异反转 | 平常场景 → 突然异变 | 惊悚、超自然 |
| 职业背景+隐秘罪恶 | 专业身份 → 不为人知 | 刑侦、暗黑 |
| 概括观点+开门见山 | 直接亮明观点 | 干货、观点、科普 |

### 3.3 各垂类写作套路

#### 职场类
```
【开头】反常识/故事切入/身份权威
【正文】问题现象 → 分析原因 → 解决方案
【结尾】上价值（升华）
```

#### 情感类
```
【开头】痛点共鸣/故事场景/数据冲击
【正文】共情 → 故事 → 观点提炼
【结尾】金句升华
```

#### 干货类
```
【开头】权威背书/承诺价值/颠覆认知
【正文】核心观点 → 详细拆解 → 延伸
【结尾】总结回顾
```

#### 观点类
```
【开头】直接表态/先抑后扬/问题引导
【正文】观点 → 论证 → 价值延伸
【结尾】开放讨论
```

### 3.4 金句制造四法

| 方法 | 公式 | 示例 |
|------|------|------|
| 对比法 | 不是A，而是B | "使人疲惫不堪的不是远方的高山，而是鞋里的一粒沙" |
| 选择法 | 可以A，但不要B | "你可以选择安逸，但不要后悔平庸" |
| 短句强化 | 排比+短句 | "每一个优秀的人都有一段沉默的时光...我们把它叫做扎根" |
| 共鸣模板 | 所有{负面}，终将成为{正面} | "所有的苦难，终将成为财富" |

## 四、微信公众号垂类爆款模板

### 4.1 标题套路

| 套路 | 公式 | 示例 |
|------|------|------|
| 激发好奇 | 为什么{现象}？ | "为什么90%的人都做错了这件事？" |
| 观点鲜明 | 我{态度}{观点} | "我为什么不建议年轻人去大厂" |
| 数字法则 | {数字}{单位}{结果} | "3个方法，让我年入百万" |
| 对比反差 | {A}与{B}的区别 | "月薪3千与月薪3万的区别" |
| 情感共鸣 | {场景}+{情绪} | "那些深夜崩溃的年轻人" |

### 4.2 正文结构（2000-4000字）

```
【开头】100-200字：吸引+预告价值
【正文】按小标题分段，每500-800字一个主题
  - 小标题1：故事/案例
  - 小标题2：分析/观点
  - 小标题3：方法/建议
  - 小标题4：延伸/升华
【结尾】100-200字：总结+行动号召
```

### 4.3 各垂类特点

| 垂类 | 字数 | 结构特点 | 高转发要素 |
|------|------|----------|------------|
| 职场 | 2000-3000 | 案例+方法论 | 实用性、代入感 |
| 情感 | 1500-2500 | 故事+观点 | 共鸣、金句 |
| 观点 | 2000-4000 | 论证+升华 | 颠覆认知、争议性 |
| 干货 | 3000-5000 | 体系+实操 | 完整性、可执行 |

## 五、微博垂类爆款模板

### 5.1 蹭热点四要素

1. **速度**：2小时内响应最佳
2. **角度**：独特视角切入
3. **深度**：有见地的分析
4. **态度**：明确立场

### 5.2 文案结构（100-120字）

```
【开头】热点关联/观点表态
【正文】核心内容（分点/故事）
【结尾】话题标签（1-3个）
```

### 5.3 爆款套路

| 类型 | 模板 | 示例 |
|------|------|------|
| 热点借势 | {热点事件}+{观点/解读} | "刘畊宏火爆背后，是全民健身的觉醒" |
| 提问互动 | {问题}，你怎么看？ | "35岁失业，你怎么看？" |
| 数字冲击 | {数字}，{解读} | "这个数字，震惊了我" |
| 共鸣型 | {群体}+{痛点} | "打工人的心酸，谁懂" |

### 5.4 话题使用策略

- 优先选择已有热度的公共话题
- 每条1-3个话题标签
- 自创话题需有明确主题
- 超话运营：定期发布+粉丝互动

## 七、通用避坑清单

### 7.1 标题避坑

❌ 过长（小红书>20字，抖音>30字）
❌ 无关键词
❌ 纯标题党（内容不符）
❌ 使用违禁词（最、第一、顶级等）

### 7.2 内容避坑

❌ 开头拖沓（3秒抓不住注意力）
❌ 信息密度过低
❌ 缺乏情绪共鸣点
❌ 结尾无互动引导

### 7.3 平台特定避坑

| 平台 | 避坑要点 |
|------|----------|
| 小红书 | emoji过多、字数过千、无配图 |
| 微信 | 超过5000字、排版混乱、无小标题 |
| 微博 | 超过140字、无话题标签、蹭热点生硬 |
| 知乎 | 答非所问、纯营销、AI味太重 |
| 抖音 | 时长过长、完播率低、开头无聊 |
| B站 | 标题党、无弹幕互动点、更新不稳定 |

## 八、选题研究员 Agent 核心逻辑

### 8.1 选题原则

**什么火就发什么**：
1. 追踪热点话题（微博热搜、抖音热榜、小红书趋势）
2. 分析竞品爆款（同垂类头部账号最近7天爆款）
3. 挖掘用户痛点（评论区高频问题、搜索词趋势）

### 8.2 选题流程

```
热点追踪 → 垂类匹配 → 爆款模板匹配 → 选题报告
```

## 九、内容创作者 Agent 核心逻辑

### 9.1 创作原则

**根据垂类风格定内容**：
1. 根据平台选择对应模板
2. 根据垂类选择对应风格
3. 套用爆款公式填充内容

### 9.2 创作流程

```
选题报告 → 模板匹配 → 内容填充 → 风格适配 → 输出草稿
```

### 9.3 质量检查

- [ ] 标题是否符合平台规范
- [ ] 开头3秒是否抓人
- [ ] 正文结构是否清晰
