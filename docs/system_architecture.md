# Crew Media Ops 系统架构文档

> Multi-Agent 自媒体运营系统 - 完整架构与 Workflow 说明

---

## 1. 系统概述

Crew Media Ops 是基于 CrewAI 的自媒体运营 Multi-Agent 系统，实现从选题、创作、发布到数据分析的全链路自动化。

**核心特性**：
- 支持 6 个国内平台：小红书、微信公众号、微博、知乎、抖音、B站
- 支持 5 个海外平台：Reddit、Twitter/X、Instagram、Facebook、Threads
- 两种发布模式：Agent 模式（LLM 驱动）和 Direct 模式（无 LLM 开销）
- 定时调度系统：支持一次性定时和 Cron 表达式
- 爆款对标研究：自动分析爆款内容结构

## 3. 核心工作流程

### 3.1 内容创作流程 (ContentCrew)

## 6. 核心数据结构

### 6.1 PublishContent

```python
class PublishContent:
    title: str                    # 标题
    body: str                     # 正文内容
    content_type: ContentType     # 内容类型 (ARTICLE, IMAGE_TEXT, VIDEO)
    images: list[str]             # 图片路径列表
    tags: list[str]               # 标签
    cover_image: str | None       # 封面图
    custom_fields: dict           # 平台特定字段
```

### 6.2 PublishResult

```python
class PublishResult:
    success: bool                 # 是否成功
    content_id: str | None        # 内容 ID
    content_url: str | None       # 内容 URL
    status_detail: str            # 状态详情
    error: str | None             # 错误信息
    data: dict                    # 额外数据
    published_at: datetime | None # 发布时间
```
