## 1. 小红书

### 1.1 爆款标题公式

| 模板类型 | 公式 | 示例 |
|---------|------|------|
| 数字型 | 数字 + 核心关键词 + 价值点 | "7个让你皮肤起死回生的护肤技巧" |
| 揭秘型 | 【揭秘】+ 领域 + 内幕/秘密 | "【揭秘】小红书爆款背后的运营逻辑" |
| 痛点型 | 具体问题 + 数字承诺 | "熬夜脸怎么救？3天恢复透亮肌" |
| 合集型 | 数字 + 行业/领域 + 必备 | "10款学生党必入平价好物合集" |

**高频开场白**：姐妹们！、宝子们～、家人们

**标题规范**：
- 字数 ≤20 字
- 前 18 字包含 2 个核心关键词
- 适当使用 emoji（1-3 个）
- 3 秒内传递核心价值

### 1.2 正文结构模板（五段式）

```
【痛点切入】引发共鸣
    ↓
【解决方案】分点列出（3-5 点）
    ↓
【效果验证】图文证明/个人经历
    ↓
【价值总结】提炼要点
    ↓
【互动引导】开放式提问（非诱导）
```

**排版要点**：
- 每 3-4 行分段
- 每 300 字重复目标关键词
- 用 emoji 分隔关键信息
- 同一表情不重复使用

**字数控制**：500-1000 字（小红书用户习惯快速浏览）

### 1.3 违禁词列表

#### 绝对禁止（广告法违禁词）

**含"最"字类**：最、最佳、最具、最优、最好、最高、顶级

**含"一"字类**：第一、唯一、首个、独一无二

**含"级/极"字类**：国家级、世界级、顶级、极品、极致

#### 高风险限流词

| 类别 | 示例 |
|------|------|
| 站外引流 | 微信号、微博、手机号、链接、天猫、淘宝、京东、联系我 |
| 诱导互动 | 直接写"点赞收藏关注"、"评论有惊喜"、"留言领取" |
| 功效型 | 美白、抗衰、消炎杀菌、祛痘、祛湿（需特证） |
| 营销味过强 | 剁手、回购无数次、下单、活动福利、相当于不要钱 |
| 迷信类 | 算命、算卦、保佑、化太岁、佛牌 |

### 1.4 图片规范

**尺寸**：
- 竖版（推荐）：3:4，1080×1440px
- 方版：1:1，1080×1080px

**封面风格**：杂志留白风、氛围感风、色块冲击风、拼贴风、手写风

**数量**：3-9 张图，首图最关键

### 1.5 发布时间

| 时段 | 时间 | 用户行为 |
|------|------|----------|
| 早高峰 | 7:00-9:00 | 通勤路上 |
| 午高峰 | 12:00-14:00 | 午休时间 |
| 晚高峰 | 19:00-22:00 | 活跃度最高 |
| 深夜黄金 | 21:00-23:00 | 流量最佳 |

**发布频率**：每周 3-4 篇优质笔记

### 1.6 算法偏好（2025-2026）

- **CES 模型升级**：权重转向"心智停留"
- **深度行为权重提升**：截图、停留时长
- **热点内容推荐权重强化**

## 2. 微信公众号

### 2.1 标题套路

| 技巧 | 示例 |
|------|------|
| 激发好奇心 | "为什么90%的人都做错了这件事？" |
| 观点鲜明 | "我为什么不建议年轻人去大厂" |
| 数字法则 | "3个方法"、"10万+背后的秘密" |
| 对比反差 | "月薪3千与月薪3万的区别" |
| 情感共鸣 | 触达读者痛点的场景描述 |

**字数**：20-30 字（建议），不超过 64 字

### 2.2 正文结构

**推荐长度**：2000-4000 字（不超过 5000 字）

```
开头（吸引 + 预告）100-200 字
    ↓
正文（干货 + 故事）按小标题分段
    ↓
结尾（总结 + 行动号召）100-200 字
```

**排版规范**：
- 正文字号：14-15px
- 行间距：1.5-1.75 倍
- 颜色不超过 3 种
- 每 500-800 字设置小标题

### 2.3 封面图规格

- 头条大图：900×383px（2.35:1）
- 头条小图：383×383px（1:1）
- 次条封面：200×200px（1:1）

### 2.4 违禁内容

**六大禁区**：政治敏感、色情低俗、暴力恐怖、虚假信息、侵权抄袭、诱导分享

**营销敏感词**：
- 支付类：账号、卡号、打款、支付宝、银行卡
- 价格类：多少钱、价格、免费领取
- 联系类：加微信、扫码、私信
- 诱导类：分享后查看、转发领取、集赞

### 2.5 发布时间

**黄金时段**：21:00-22:00
**次优时段**：7:00-9:00、12:00-14:00

**频率**：每天可群发 1 次，最多 8 篇文章

### 2.6 算法机制

**看一看推荐（2024+）**：
- 订阅+推荐混合模式
- 核心指标：阅读完成率
- "朋友在看"权重上升

## 3. 微博

### 3.1 内容特征

**文体核心**：短平快，140 字限制（建议 100-120 字）

**话题使用**：
- 每条 1-3 个话题标签
- 优先选择已有热度的公共话题
- 自创话题需有明确主题

**超话运营**：
- 选择与账号定位相符的超话
- 定期发布优质内容
- 与粉丝互动提升超话等级

### 3.2 蹭热点四要素

1. **速度**：2 小时内响应
2. **角度**：独特视角
3. **深度**：有见地分析
4. **态度**：明确立场

### 3.3 违禁词

**社区公约 9 类违规**：
- 垃圾广告
- 辱骂攻击
- 诱导投票
- 重复发布
- 刷粉刷赞
- 人肉搜索
- 不友善内容
- 低俗内容
- 抄袭侵权

**营销号红线**：搬运抄袭、标题党、刷量刷粉、频繁广告

### 3.4 算法机制

**热搜热度公式**：（搜索热度 + 讨论热度 + 传播热度）× 互动率

**2025 热搜配比**：
- 社会时政 30%
- 垂直热点 30%
- 文娱 30%

### 3.5 发布时间

- 早高峰：7:00-9:00
- 午休：12:00-14:00
- 晚高峰：18:00-22:00（流量最大）

## 4. 知乎

### 4.1 回答 vs 文章

| 维度 | 知乎回答 | 知乎文章 |
|------|---------|---------|
| 目的 | 针对具体问题提供解决方案 | 独立表达，自由选题 |
| 入口 | 必须依附于某个问题 | 独立发布 |
| 长度 | 较短，够用即可 | 可更长（1-3 万字） |

### 4.2 写作技巧

**专业感与通俗度平衡**：
- 专业内容用通俗语言表达
- 情绪价值 + 干货价值结合
- 使用故事记叙、情绪共鸣

**排版要点**：
- 段落控制在 3-5 句话
- 使用小标题、加粗、引用块
- 图文比例舒服

### 4.3 违禁内容

**内容被折叠原因**：
- 答非所问
- 违反社区规范
- AI 批量生成内容
- 质量低下
- 营销推广过重

**广告识别标准**：
- 品牌产品频繁露出
- 内容过度夸张
- 拿钱写的宣传文章

### 4.4 盐值系统

| 维度 | 最高分 |
|------|--------|
| 基础信用 | ~200 分 |
| 内容创作 | ~200 分 |
| 友善互动 | ~200 分 |
| 遵守规范 | ~200 分 |
| 社区建设 | ~200 分 |

**权益等级**：
- 600 分：审核优先
- 620 分：编辑问题
- 750 分：评论区发图

### 4.5 2025-2026 趋势

- 重点扶持"行业专家 IP + 垂直内容矩阵"
- 优质创作者补贴提升至 1.5 倍
- 矩阵铺量已失效，精耕细作才是出路

## 5. 抖音

### 5.1 黄金 3 秒原则

前 3 秒决定视频生死：

**实操策略**：
- **设置冲突**：开头即抛出矛盾点
- **制造悬念**：用"你绝对想不到..."、"结果..."
- **价值钩子**：直接告诉用户看完能获得什么

### 5.2 脚本结构（黄金 15 秒）

| 时间 | 内容要素 | 目的 |
|------|---------|------|
| 前 3 秒 | 冲突/悬念 | 抓住注意力 |
| 第 5 秒 | 提问引导 | 激发好奇心 |
| 第 10 秒 | 福利/价值承诺 | 提升完播率 |

**时长建议**：15-60 秒（低粉爆款榜 90% 在此时长）

### 5.3 违禁词

**绝对化用语**：
- 极限词：最、第一、顶级、极品、首家
- 时间词：史无前例、永久、绝对、100%
- 级别词：国家级、世界级、最高级

**医疗健康类**：治疗、治愈、疗效、消炎、抗癌

**金融投资类**：保本、稳赚、零风险、高收益

### 5.4 算法机制

**流量池层级**：

| 等级 | 曝光量 | 触发条件 |
|------|--------|---------|
| 冷启动 | 200-500 次 | 新视频发布 |
| 初级 | 1,000-5,000 次 | 冷启动数据良好 |
| 中级 | 1 万-10 万次 | 初级数据优秀 |
| 高级 | 10 万+ 曝光 | 中级数据爆发 |
| 热门 | 百万级曝光 | 全站爆款 |

**指标权重**：

| 指标 | 权重 |
|------|------|
| 完播率 | ~40% |
| 点赞率 | ~20% |
| 转发率 | ~15% |
| 评论率 | ~10% |
| 关注率 | ~10% |
| 复播率 | ~5% |

### 5.5 2026 内容运营核心

- **"软文为王，钩子定生死"**
- 将长内容拆解为多个短视频
- 前 1-2 小时表现至关重要

## 6. B站

### 6.1 内容特征

**视频类型**：
- 知识科普
- 游戏解说
- 鬼畜二创
- VLOG 生活
- 音乐舞蹈

**弹幕文化**：与观众实时互动，增强社区感

### 6.2 算法推荐

**分阶段推送**：

1. **初始测试期**（3-7 小时）
   - 进入小范围测试池
   - 评估完播率、点赞率、弹幕互动

2. **核心爆发期**（7-13 小时）
   - 完播率 >40%、点赞率 >5% 可进入大推荐池
   - 获得流量爆发

3. **长尾推荐期**
   - 10% 优质内容获 1-30 天二次推荐
   - 1% 冷门垂类可能被"考古"

**核心指标**：完播率、弹幕互动、点赞分享、评论数

### 6.3 违禁内容

**禁止行业**：
- 赌博类、彩票、比特币
- 烟草类、走私盗版

**形象限制**：
- 不得使用国家机关工作人员形象
- 包含真人及动漫形象

**公序良俗**：不得违反社会公德

### 6.4 2025 新趋势

- 优先推送"正能量"内容（科技、教育、生活）
- 强调"价值共创"
- 新增"此类内容过多"反馈选项
- 用户可管理标签偏好
