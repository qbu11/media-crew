# 小红书运营规范

> 基于对小红书平台的深入研究，汇总的运营规范与最佳实践。最后更新：2026-03-26

---

## 一、爆款标题公式

### 1. 标题基本原则
- **字数控制在20字以内**，尽量简短
- **前18字必须包含2个核心关键词**
- **3秒内传递核心价值**
- **标题中包含emoji表情符号**，增加活力
- **口语化表达**，拉近与读者距离

### 2. 爆款标题公式模板

| 模板类型 | 公式 | 示例 |
|---------|------|------|
| **数字型** | 数字 + 核心关键词 + 价值点 | "7个让你皮肤起死回生的护肤技巧" |
| **揭秘型** | 【揭秘】+ 领域 + 内幕/秘密 | "【揭秘】小红书爆款背后的运营逻辑" |
| **痛点型** | 具体问题 + 数字承诺 | "熬夜脸怎么救？3天恢复透亮肌" |
| **合集型** | 数字 + 行业/领域 + 必备/推荐 | "10款学生党必入平价好物合集" |
| **可视化卖点** | 价格/数量 + 产品 + 效果 | "均价20+平价好物分享" |
| **痛点解决** | 痛点 + 解决方案 | "姐妹们！这个方法拯救了我的皮肤" |
| **正负刺激** | 正面/负面情绪词 + 核心内容 | "千万别买！vs 必须拥有！" |
| **符号点睛** | 核心内容 + 视觉符号 | 使用❗️⚠️等符号增强视觉冲击力 |

### 3. 常用开场白/称呼
- **姐妹们！** - 快速拉近与读者距离，营造亲切分享氛围
- **宝子们～** - 亲切感强，适合年轻用户群体
- **家人们** - 社群归属感
- **集美们** - 闺蜜感

### 4. 高频情绪词
- 正面：必看、火爆、绝绝子、yyds、宝藏、惊喜、推荐
- 负面（慎用）：踩雷、后悔、避坑、千万别

## 二、正文结构模板

### 1. 标准五段式结构

```
【痛点切入】引发共鸣
例："你是不是也总找不到适合自己的口红？"
      ↓
【解决方案】分点列出
• 推荐产品/方法 + 真实体验
• 正文每300字重复目标关键词一次
      ↓
【效果验证】图文证明
• 前后对比图/使用场景图
• 真实数据支撑
      ↓
【价值总结】提炼要点
• 用emoji分隔关键信息
• 每3-4行分段，提高可读性
      ↓
【互动引导】促发评论
例："你最喜欢哪一款？评论区告诉我！"
```

### 2. 开头钩子写法

| 钩子类型 | 说明 | 示例 |
|---------|------|------|
| **犀利提问** | 用问题引发思考 | "你是不是也有这个问题？" |
| **惊人结论** | 反常规结论吸引眼球 | "原来这么多年都做错了！" |
| **效果展示** | 直接展示使用前后对比 | "30天变化，判若两人" |
| **痛点共鸣** | 直击用户痛点 | "姐妹们！这个方法拯救了我" |
| **互动挑战** | 设置挑战引发关注 | "我来挑战30天..." |

**关键原则**：开头5秒能留住人，笔记就成功了一半。

### 3. 结尾互动引导

**合规引导方式**：
- 设计互动话题，引导用户留下有价值的**长评论**
- 提问引发评论区讨论
- 设置悬念预告下期内容

**违规引导方式（禁止）**：
- 直接写"点赞收藏关注"
- 用符号代替文字进行引导（如 👍⭐️❤️）
- "评论有惊喜"、"转发有奖"等诱导性文字

### 4. 6大爆款笔记类型

| 类型 | 结构 | 适用场景 |
|------|------|----------|
| **干货合集** | 痛点 → 分点干货 → 总结 | 教程、攻略类 |
| **种草测评** | 引入需求 → 产品介绍 → 体验感受 → 推荐 | 产品推广 |
| **避坑雷区** | 踩坑经历 → 问题分析 → 正确做法 | 警示类内容 |
| **对比测评** | 多品对比 → 优缺点 → 结论 | 选品指南 |
| **教程攻略** | 步骤拆解 → 图文演示 → 成果展示 | 操作类内容 |
| **情感故事** | 故事引入 → 情感共鸣 → 价值升华 | 个人IP |

### 5. 排版规范

- **多用空行凸显层次**：80%的新手博主容易忽略空行的重要性
- **分段式写作**：用小标题、空行、emoji隔开不同区块
- **每3-4行分段**：增加可读性
- **关键信息用emoji分隔**

## 三、Emoji表情符号使用指南

### 1. 常用表情类型

| 类型 | 常用表情 | 用途 |
|------|---------|------|
| **装饰性** | 😊🎀✨💕 | 增添趣味与情感 |
| **强调类** | ❗️⚠️❌✅ | 突出重点信息 |
| **数字标号** | ①②③➊➋➌ | 分点列举 |
| **可爱符号** | ꒰ᐢ. .ᐢ꒱ ♡‧₊˚ | 装饰点缀 |
| **情绪表达** | 😭😍🥺🔥 | 情绪渲染 |

### 2. 使用原则
- 每个段落适当添加emoji，但**同一表情不要多次重复**
- 用于表示顺序、情绪或丰富文本内容
- 配合分段排版提升阅读体验
- **标题中适量使用**，正文段落中适当点缀

## 四、话题标签使用技巧

### 1. 标签数量
- **每篇笔记建议使用10个以内的标签**（官方限制不超过10个）
- 超过10个标签流量会受到限制
- 在10个以内是越多越好，能最大化曝光

### 2. 标签选择策略

| 策略 | 说明 |
|------|------|
| **研究热门标签** | 进入首页，关注热门话题 |
| **相关性原则** | 标签应与笔记内容有强烈相关性 |
| **目标受众匹配** | 根据目标受众选择标签 |
| **利用系统推荐** | 发布时使用系统推荐标签 |

### 3. 标签类型搭配
- **1-2个大流量标签**：如 #必入好物# #居家神器#
- **3-5个垂直标签**：与内容直接相关
- **2-3个精准长尾标签**：小众但精准

## 五、违禁词/敏感词列表

### 1. 绝对禁止词汇

#### 广告法违禁词（绝对化用语）
| 类别 | 违禁词 |
|------|--------|
| **含"最"字类** | 最、最佳、最具、最爱、最赚、最优、最优秀、最好、最大、最高、最高级、最高端、最奢侈、最棒、最强、超强 |
| **含"一"字类** | 第一、唯一、首个、首选、独一无二 |
| **含"级/极"字类** | 国家级、世界级、顶级、极品、极致 |
| **含"首/家/国"字类** | 首个、首家、国家级、世界首发 |
| **品牌相关** | 领导品牌、世界领先、行业领先 |
| **权威相关** | 质量免检、国家认证、认证级 |
| **时间相关** | 永远、永久 |

### 2. 高风险限流词

#### 站外引流类
- 地球号、微信号、微博、手机号、链接、天猫、淘宝、京东、APP名称、联系我、加我

#### 诱导互动类
- 点赞、收藏、关注、评论、私信、私我（直接引导时）

#### 功效型词汇（化妆品/护肤领域）
- 美白、抗衰、消炎杀菌、祛痘、祛湿、治疗、治愈、医学专业词汇

#### 营销味过强
- 剁手、回购无数、下单、活动福利、相当于不要钱、不买不是人

#### 迷信类
- 算命、算卦、保佑、玉皇大帝、鬼怪、如来、化解小人、化太岁、佛牌

#### 人身攻击类
- 装X、撕X、傻X、绿茶婊等侮辱性词汇

### 3. 敏感词检测工具
- **句易网**（https://www.ju1.cn/）- 检测标题、正文、封面敏感词
- **小红书官方审核** - 发布前自检

### 4. 敏感词处理原则
- **检测工具结果仅供参考**，需结合语境判断
- 例如"这是我第一次装修"中的"第一"不算敏感词
- 但"这个奶粉的销量全网第一"中的"第一"违反广告法

## 六、图片风格要求

### 1. 尺寸规范

| 比例 | 尺寸 | 推荐程度 |
|------|------|----------|
| **竖版（推荐）** | 3:4，1080×1440px | 最推荐，屏幕占比大 |
| **方版** | 1:1，1080×1080px | 适中 |
| **横版** | 4:3，1440×1080px | 较少使用 |

### 2. 5大爆款封面风格

| 风格类型 | 特点 |
|---------|------|
| **杂志留白风** | 简洁大气，留白空间多，高级感强 |
| **氛围感风** | 注重情绪营造，色调柔和统一 |
| **色块冲击风** | 高饱和度色块拼接，视觉冲击力强 |
| **拼贴风** | 多图组合，内容丰富有趣 |
| **手写风** | 手写字体，亲切自然，个性化 |

### 3. 封面设计要点

| 要素 | 要求 |
|------|------|
| **背景** | 单色、渐变色或质感照片，简洁不抢眼 |
| **标题文字** | 大字号（60pt以上），清晰可读 |
| **配色** | 高饱和度色块或协调配色 |
| **信息传达** | 第一眼就能抓住用户，快速了解主题 |
| **风格统一** | 与账号定位和内容调性一致 |

### 4. 图片质量要求
- **必须原创**，无水印
- **清晰美观**
- **主题突出**
- **色彩鲜明**

| 时段 | 时间 | 用户行为特征 |

| **早高峰** | 7:00-9:00 | 通勤路上刷手机，碎片化浏览 |
| **午高峰** | 12:00-14:00 | 午休时间，吃饭休息时刷内容 |
| **晚高峰** | 19:00-22:00 | 下班放松时间，浏览活跃度最高 |
| **深夜黄金档** | 21:00-23:00 | 躺在沙发上刷，流量最佳 |

### 2. 不同人群精准发布时间

| 目标人群 | 推荐发布时间 |
|---------|-------------|
| **上班族** | 工作日 6:30-7:00（通勤黄金时间）|

### 3. 发布策略

| 策略 | 说明 |
|------|------|
| **常规策略** | 在流量高峰期发布，获得更多初始曝光 |
| **差异化策略** | 避开高峰期，选择15:00-16:00或11:00-11:30，减少竞争 |

### 4. 发布频率建议
- **每周发布3-4篇优质笔记**
- 保持账号活跃度
- **避免频繁删除笔记**（每月超过5篇会降权）

## 八、算法推荐机制

### 1. 2025年算法核心变化
- **CES（内容体验分）模型升级**
- **新算法核心权重转向"用户停留"背后的"心智停留"**
- 例如：一秒的截图动作、页面停留时长等深度行为
- **热点权重强化**：2025年进一步强化热点内容的推荐权重

### 2. "奥利奥模式"流量分配
- **先人感，后算法，再人感**的动态放量模型
- 冷启期进行白盒投放
- 企业可针对目标受众进行自定义

### 3. 互动权重差异
- 不再单纯依赖点赞、收藏、评论等浅层互动
- 深度行为（截图、停留时长）权重提升

## 十、运营自检清单

### 发布前检查
- [ ] 标题是否包含核心关键词（前18字2个）
- [ ] 标题字数是否在20字以内
- [ ] 正文是否使用了敏感词检测工具
- [ ] 封面图片是否清晰美观、无水印
- [ ] 是否选择了10个以内相关话题标签
- [ ] 结尾是否有合规的互动引导
- [ ] 发布时间是否在流量高峰期

### 内容质量检查
- [ ] 开头是否能在5秒内抓住用户
- [ ] 中间是否分层提供足够价值
- [ ] 结尾是否有明确的互动或行动指令
- [ ] 排版是否每3-4行分段
- [ ] 是否适当使用emoji表情
