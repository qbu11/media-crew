# 小红书笔记

## 标题
AI编程工具大爆发！这3个GitHub项目一夜涨粉数千

## 正文
最近 GitHub Trending 被 AI Agent 工具霸榜了，分享 3 个值得关注的项目👇

1️⃣ superpowers
给 AI 编程助手装"技能包"的框架。你可以用自然语言定义技能，让 Claude Code 等工具自动学会新能力。Shell 实现，极简但实用。

2️⃣ oh-my-claudecode
Claude Code 的多 Agent 编排工具。一个人指挥多个 AI 同时干活，写代码、跑测试、做 review 全自动。Teams-first 设计，适合复杂项目。

3️⃣ dexter
自主金融研究 Agent。给它一个问题，它会自己搜索、分析、写报告。TypeScript 实现，开箱即用。

这三个项目说明一个趋势：AI 编程正在从"一个助手"变成"一支团队"。

以前是你问 AI 一个问题，它回答你。
现在是你给 AI 一个目标，它自己拆解任务、分配角色、协作完成。

Multi-Agent 不再是论文里的概念，而是真正可用的开发工具了。

💡 如果你在用 Claude Code 或 Cursor，强烈建议试试 superpowers，给你的 AI 助手加技能真的很爽。
