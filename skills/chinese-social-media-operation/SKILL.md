# Chinese Social Media Operation Skill

> 中文社交媒体运营专家 - 掌握 6 大平台运营规范、爆款逻辑和违禁词检测

---

## 触发词

`运营规范`、`平台违禁词`、`内容审核`、`爆款标题`、`小红书运营`、`微信运营`、`微博运营`、`知乎运营`、`抖音运营`、`B站运营`

---

## 功能描述

本 Skill 提供以下功能：

1. **平台运营规范查询**：快速查询各平台的运营规范、内容特征、发布时间
2. **违禁词检测**：检测内容中的广告法极限词、诱导互动词、敏感内容
3. **爆款标题生成**：根据平台特点生成高传播力标题
4. **内容风格适配**：将内容适配到不同平台的风格和格式

## 平台速查表

### 小红书

| 项目 | 规范 |
|------|------|
| 标题长度 | ≤20字，前18字含2个核心关键词 |
| 正文长度 | 500-1000字 |
| 标题模板 | 数字型/揭秘型/痛点型/合集型 |
| 开场白 | 姐妹们！、宝子们～、家人们 |
| 发布时间 | 21:00-23:00最佳 |
| 发布频率 | 每周3-4篇 |
| emoji使用 | 1-3个，用emoji分隔关键信息 |

### 微信公众号

| 项目 | 规范 |
|------|------|
| 标题长度 | 20-30字（建议），不超过64字 |
| 正文长度 | 2000-4000字（不超过5000字） |
| 标题套路 | 激发好奇心/观点鲜明/数字法则/对比反差 |
| 发布时间 | 21:00-22:00黄金时段 |
| 发布频率 | 每天1次，最多8篇 |
| 封面图 | 头条大图900×383px，小图383×383px |

### 微博

| 项目 | 规范 |
|------|------|
| 文体 | 短平快，100-120字 |
| 话题标签 | 每条1-3个 |
| 蹭热点 | 2小时内响应 |
| 发布时间 | 18:00-22:00流量最大 |

### 知乎

| 项目 | 规范 |
|------|------|
| 回答长度 | 较短，够用即可 |
| 文章长度 | 1-3万字 |
| 写法 | 专业+通俗平衡 |
| 盐值权益 | 600分审核优先，750分评论区发图 |

### 抖音

| 项目 | 规范 |
|------|------|
| 黄金3秒 | 开头抛冲突/悬念/价值钩子 |
| 视频时长 | 15-60秒 |
| 核心指标 | 完播率~40% |
| 流量池 | 冷启动(200-500)→初级(1K-5K)→中级(1W-10W)→高级(10W+) |

### B站

| 项目 | 规范 |
|------|------|
| 推送阶段 | 初始测试(3-7h)→核心爆发(7-13h)→长尾推荐 |
| 核心指标 | 完播率、弹幕互动、点赞分享 |
| 爆发条件 | 完播率>40%、点赞率>5% |

## 使用方式

### 作为 Claude Code Skill

直接在对话中提及触发词，如：
- "帮我检查这段内容有没有违禁词"
- "生成一个小红书爆款标题"
- "把这篇内容适配成抖音脚本"

### 作为 Python 模块

```python
from src.agents.content_reviewer import ForbiddenWordsChecker

# 检查违禁词
result = ForbiddenWordsChecker.check_all("你的内容", platform="xiaohongshu")

# 获取合规分数
score = ForbiddenWordsChecker.get_compliance_score(result)

# 生成优化建议
suggestions = ForbiddenWordsChecker.generate_suggestions(result)
```
