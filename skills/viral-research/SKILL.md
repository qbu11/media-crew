# 选题研究 Skill

> 核心逻辑：什么火就发什么

---

## 触发词

`选题`、`爆款分析`、`热点追踪`、`竞品分析`、`内容策划`、`垂类选题`

---

## 选题原则

**什么火就发什么**：
1. 追踪热点话题（微博热搜、抖音热榜、小红书趋势）
2. 分析竞品爆款（同垂类头部账号最近7天爆款）
3. 套用成功模板（标题公式、结构模板、爆款要素）

---

## 选题流程
```
热点追踪 → 垂类匹配 → 爆款模板匹配 → 选题报告
```

---

## 热点追踪来源
- 微博热搜榜（2小时内响应最佳）
- 抖音热榜（关注上升趋势）
- 小红书热门笔记（找同垂类爆款）
- 知乎热榜（高赞回答分析）

## 各垂类爆款标题模板

### 美妆护肤类
| 模板 | 公式 | 示例 |
|------|------|------|
| 肤质+痛点+方案 | {肤质}+{痛点}+{解决方案} | "混油敏皮闭口反复？这个成分救了我" |
| 数字+效果 | {数字}天+{效果描述} | "30天逆袭冷白皮，亲测有效" |
| 对比反差 | 别再X了，试试Y | "别再刷酸了！这个才是敏皮救星" |

### 职场成长类
| 模板 | 公式 | 示例 |
|------|------|------|

### 穿搭类

## 正文结构模板
### 干货攻略类
```
【开头】痛点共鸣 + 承诺价值
【正文】方法1/2/3（步骤+原理）
【结尾】总结 + 互动引导
```

### 种草测评类
```
【开头】使用场景/痛点 + 产品引入
【正文】外观/感受/效果对比
【结尾】适合人群 + 购买建议
```

### 避坑合集类
```
【开头】踩坑经历 + 引发共鸣
【正文】❌ 误区 + ✅ 正确做法
【结尾】总结 + 评论区互动
```
